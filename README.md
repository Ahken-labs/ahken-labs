
# Ahken Labs - Company Website

Official website for Ahken Labs, built using Next.js with TypeScript, Tailwind + css, featuring reusable components, smooth animations, and scalable architecture.

- This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).


## Running Locally
### 1. Clone the Repository

```bash
git clone <repository-url>
cd Ahken-Labs
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Set Up Environment Variables

Create a `.env` file based on `.env.example`:

```bash
cp .env.example .env
```

Update the values in `.env` as needed for your environment.

### 4. Run the Application

```bash
# Development mode with auto-restart
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.


## Deploy on Vercel

This website is deployed on vercel. available on https://ahken-labs.vercel.app/ .
